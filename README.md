# OpenWRT USB to Internal Disk Installer (x86)

Скрипт для автоматической установки OpenWRT со съемных носителей на диск x86_x64-совместимого компьютера (SSD/HDD).  
Скрипт сам находит исходный диск с OpenWRT, создаёт разделы, копирует систему и настраивает загрузчик.

## ✨ Возможности

- 🌍 **Двуязычный интерфейс** (русский/английский) — выбор языка в начале
- 🌐 **Проверка интернета** — скрипт проверяет доступность `1.1.1.1`, `9.9.9.9`, `openwrt.org`, `kernel.org`
- 💾 **Автоматическое создание разделов** (sfdisk):
  - GPT таблица разделов
  - EFI раздел (256 МБ, FAT32) с правильным GUID и флагом ESP
  - DATA раздел (ext4) на всём оставшемся месте
- 🔍 **Умный поиск исходного диска** — ищет OpenWRT по наличию `openwrt_release`, `os-release`, `banner`, директорий `apk`/`opkg` и другим признакам
- 🔄 **Два метода копирования**:
  - `rsync` (приоритет) — показывает итоговый размер скопированных данных
  - `cp` (запасной вариант) — если rsync недоступен
- ⚙️ **Обновление PARTUUID** в `grub.cfg` для корректной загрузки с нового диска
- 📝 **Полное логирование** всех действий
- 🎨 **Цветной и структурированный вывод** — пустые строки отделяют логические блоки
- ⏳ **Индикатор прогресса** при очистке (спиннер)
- ✅ **Итоговая статистика** и понятные подсказки для новичков (клавиши входа в Boot Menu)

---

## 📦 Требования

- Компьютер с архитектурой **x86_64**
- **OpenWRT**, установленный на съемном носителе (USB-флешка) — именно с него должна быть выполнена загрузка
- **Внутренний диск** (SSD/HDD) для установки
- **(Опционально)** доступ в интернет для автоматической установки дополнительных пакетов (`sfdisk`, `rsync` и др.)

## 🚀 Установка и запуск

1. **Загрузитесь** со съемного носителя (USB-флешки), на котором уже установлен OpenWRT.
2. **Подключитесь** по SSH или откройте терминал прямо на устройстве.
3. **Скачайте и запустите скрипт** одной командой:

```bash
wget -O install.sh https://raw.githubusercontent.com/Rever-end/openwrt_usb_installer_x86/main/install.sh
chmod +x install.sh
./install.sh
```

1. Следуйте инструкциям на экране:
   · Выберите язык (1 — русский, 2 — английский)
   · Выберите целевой диск для установки (все данные на нём будут уничтожены!)
   · Подтвердите выбор, введя yes
   · Подтвердите исходный диск (скрипт найдёт его автоматически)
   · Дождитесь завершения копирования и настройки
   · Перезагрузитесь и выберите новый диск в Boot Menu BIOS/UEFI

🔄 Процесс работы

1. Проверка системы — определение менеджера пакетов (apk/opkg), проверка интернета.
2. Установка пакетов — при наличии интернета устанавливаются sfdisk, dosfstools, rsync, blkid, nano, parted, mount-utils.
3. Выбор целевого диска — показываются все реальные диски (исключая loop, ram, CD-ROM).
4. Поиск исходного диска — сканирование всех разделов в поисках характерных файлов OpenWRT.
5. Создание разделов на целевом диске (GPT, 256MB EFI + DATA на остаток).
6. Форматирование (FAT32 для EFI, ext4 для DATA) и установка флага ESP.
7. Копирование системы (приоритет — rsync с показом итогового размера, запасной вариант — cp).
8. Обновление PARTUUID в конфигурации загрузчика.
9. Очистка (размонтирование, удаление временных файлов) с индикатором прогресса.
